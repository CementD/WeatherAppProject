# WeatherAppProject

## Встановлення

1. Клонуйте репозиторій
2. Встановіть залежності: `npm install`
3. Створіть файл `.env` та додайте ваш API ключ
4. Запустіть проєкт: `npx expo start`

## Використані технології

- React Native
- TypeScript
- Expo
- OpenWeatherMap API
